Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are elements nested inside other elements.
  • Parent elements are elements that hold all of the children.
  • Semantic HTML elements are elements in which the name describes the content and the role it plays on the web page.
  • HTML elements are case sensitive.
  • Head elements are not visible to the user, body elements are.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS gives the content of a webpage style.
  • The style metatag defines the CSS styles associated with a website.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External CSS style sheet is the most common way to style a webpage.
  • Inline CSS is used when a developer wants to style an element directly in the HTML file.
  • Internal Style Sheet is used for embedding an entire stylesheet directly into the HTML file using a style element in the head element.

Git

  • Git is a free open source version control system that lets web developers collaborate on projects without overwriting each others work.
  • GitHub is the largest provider of internet hosting for software and web development code.
  • Every project on GitHub is called a Repository or "Repo"
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: checks what branch we are currently on

JavaScript

  • JavaScript (JS) handles logic and allows users to interact with a website.
  • JavaScript is a programming language used to provide logic and interactivity for websites.
